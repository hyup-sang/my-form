# My Forms
간단한 설문 양식을 생성하고 설문 조사, 결과 요약 등을 제공합니다.

개인 공부용으로 **Frontend는 간단한 화면만 제공하며, Backend 위주**로 작업합니다.

## 기술 스택
- Java 17
- Spring Boot
- Spring JPA
- React

## git branch

- api - backend
- front - frontend
